DOM탐색메서드의 종류

1.getElementById("idName"):태그의 id속성 이름으로 탐색 - 반환값은 엘리먼트 1개

ID:test1

2.getElementsByName("name속성이름"): input 태그의 name속성이름으로 선택 - 반환값은 엘리먼트 여러개(배열)

3.getElementsByTagName("tag이름"):태그 이름으로 탐색 - 반환값은 엘리먼트 여러개(배열)
태그이름탐색

4.getElementsByClassName("class이름"):class속성 이름으로 탐색 - 반환값은 엘리먼트 여러개(배열)
class이름탐색

5.querySelectorAll("selector표현식"):반환값 여러개
querySelector("selector표현식"):반환값 한개
selector쿼리탐색
